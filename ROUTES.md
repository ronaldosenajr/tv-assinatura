## Rotas da API

### Clientes

| Verbo  | Rota         | Ação              |
| ------ | ------------ | ----------------- |
| GET    | /clients     | Listar clientes   |
| GET    | /clients/:id | Mostrar cliente   |
| POST   | /clients     | Criar cliente     |
| PATCH  | /clients/:id | Atualizar cliente |
| DELETE | /clients/:id | Remover cliente   |
